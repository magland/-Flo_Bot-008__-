Trigger word:            Description:                               Syntax:

!help                    Displays the screen.                       -
!hotchocolate            Gives you a warm cup of hot chocolate.     -
!hello                   Gives a greeting.                          -
!tea                     Gives you a hot cup of tea.                -
!pizza                   Orders you a pizza.                        -
!rules                   Displays the server rules.                 -
!allquotes               Displays a list of all the quotes.         -
!randomquote             Displays a random quote.                   -
!addquote                Allows user to add a quote.                !addquote ‘quote’
!randomnumber            Displays a random number in the range      !randomnumber min max
!count                   Counts upward in 1sec intervals up to
                         desired number.                            !count tonumber
!welcome                 Displays welcome message to specified
                         user.                                      !welcome user
!punch                   Punches specified user.                    !punch user
!kiss                    Kisses specified user.                     !kiss user
/kill                    Kills specified program running.           /kill program
